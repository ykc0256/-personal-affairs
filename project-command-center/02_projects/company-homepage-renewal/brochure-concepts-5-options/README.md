# GWD / WAI 브로슈어 5가지 방향안

## 목적

GWD와 WAI 브로슈어를 각각 5가지 방향으로 나누어 검토하기 위한 작업 폴더다.

- `1안`: 사용자 의견을 반영한 현재 기준안
- `2~5안`: 과제 발표자료, 제안서, 홈페이지 리뉴얼 방향, GWD-ESAIN 비교 내용을 바탕으로 메시지 각도를 다르게 잡은 대안

## 파일 구성

| 파일 | 내용 |
| --- | --- |
| `gwd-brochure-5-options.md` | GWD 브로슈어 5가지 방향안 |
| `wai-brochure-5-options.md` | WAI 브로슈어 5가지 방향안 |
| `prompts/gwd/gwd-image-prompts-5-options.md` | GWD 5가지 안별 이미지 생성 프롬프트 |
| `prompts/wai/wai-image-prompts-5-options.md` | WAI 5가지 안별 이미지 생성 프롬프트 |

## 빠른 검토 순서

1. `gwd-brochure-5-options.md`에서 GWD 5안 중 톤 선택
2. `wai-brochure-5-options.md`에서 WAI 5안 중 톤 선택
3. 선택한 안의 프롬프트를 `prompts/`에서 복사해 이미지 시안 생성
4. 이미지 결과를 보고 문구 밀도, 도식 비중, 제목 톤 조정

## 공통 기준

- 외부 공개용 브로슈어 기준으로 작성한다.
- GWD 공식 영문 풀네임은 `Generative WWTP Design`으로 사용한다.
- WAI는 완성된 제품이나 SaaS가 아니라 `WATER AI 개념`으로 표현한다.
- 관계 흐름은 `GWD → AGWD → WAI`로 정리한다.
- AGWD는 언급 가능하되, 오믈렛·세부 AI 모듈명·협력사 수행 체계는 제외한다.
- TRL 5는 외부 공개용 브로슈어에서 제외한다.
- 원자료에 있는 `세계 유일`, `80% 단축`, `50% 절감` 표현은 사용 가능하되, 배치 위치는 시안 검토 후 결정한다.
