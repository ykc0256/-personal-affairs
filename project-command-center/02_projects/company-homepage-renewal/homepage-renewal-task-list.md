# 홈페이지 리뉴얼 업무 체크리스트

## 목적

홈페이지 리뉴얼과 관련해 최윤규 리더가 확인하고 진행해야 할 업무를 한 곳에서 관리한다.

현재 리뉴얼은 최윤규 리더가 작성하는 기술소개자료(브로셔)를 기반으로 진행할 예정이며, 기술소개자료 작성 이후 이옥주 리더님과 상세 미팅을 잡아 제작 범위와 일정을 구체화한다.

## 주요 전제

- 홈페이지 리뉴얼은 기술소개자료(브로셔)를 기반으로 제작한다.
- 기술소개자료(브로셔)는 최윤규 리더가 작성한다.
- 기술소개자료 작성 예상 기간은 `2026-05-26 ~ 2026-05-28`이다.
- 이옥주 리더님과의 상세 미팅은 기술소개자료 작성 이후 진행한다.
- 5월 말 홈페이지 리뉴얼 착수 여부는 본부장님 확인이 필요하다.
- 강수진 리더는 기술소개자료 및 홈페이지 관련 디자인을 지원한다.
- 디자인은 홍보팀 협업을 통해 구체화한다.
- TDG 브로셔(`TRANSCEND_Corporate_Deck_Aug_2022.pdf`)는 구성과 메시지 참고 자료로 활용한다.
