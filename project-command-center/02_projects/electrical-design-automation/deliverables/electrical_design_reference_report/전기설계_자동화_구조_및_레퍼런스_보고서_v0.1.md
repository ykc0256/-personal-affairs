## 보고자료용 핵심 문장

WAI Design의 전기설계 자동화는 공정 설계와 기계 카탈로그 선택값을 출발점으로 삼아, 국내 전기설계 기준과 전기/계측 카탈로그 DB를 적용하고, 최종적으로 용량계산서, 도면, 수량산출서, 내역서를 일관된 데이터 흐름으로 생성하는 구조입니다.
