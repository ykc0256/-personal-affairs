### 3. 참조표 성능 개선 및 테이블 정리

**문제:** 참조표 페이지 진입 시 렉(lag) 발생, 테이블 빈 컬럼 과다

**해결 방법:**

1. **토글 래핑** — 12개 테이블 각각을 토글 블록으로 감싸 lazy rendering 적용
2. **빈 컬럼 제거** — 헤더 행에서 첫 None 또는 '예시' 셀 이후 컬럼 모두 제거
3. **비고 통합** — 제거된 컬럼들의 값을 하나의 '비고' 컬럼으로 합산 (`' / '` 구분)
4. **비고 자동 제거** — 비고 컬럼이 전 행에서 비어 있으면 열 자체 삭제
5. **섹션 분리** — ①②③ 유니코드 원문자 행 감지, 각 섹션을 h3 소제목으로 구분
