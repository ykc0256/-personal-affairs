## 핵심 판단

전기설계 자동화는 단일 Excel 계산 자동화가 아니라, 공정 설계와 기계 카탈로그 선택 결과를 전기설계 입력값으로 받아 계산 결과를 3D 배치, 2D 도면, 케이블 스케줄, 물량/내역, PDF/DXF/Excel 출력까지 연결하는 설계 파이프라인으로 정의되어 있다.

`260106_전기계산서(최종샘플).xlsx`는 이 파이프라인의 계산 엔진과 산출물 템플릿 원형에 가깝다. `카탈로그_기계_v1.5.0.xlsx`는 전기계산서 입력값 중 동력, 기동방식, 정격전압, 효율, 역률, 수용율의 원천 데이터 역할을 한다. `250721_기능정의서.xlsx`는 이를 프로그램 기능으로 확장한 요구사항 정의서이고, `260414_WAI Design_튜토리얼 매뉴얼_1.0.1.pdf`는 WAI Design 안에서 사용자가 프로젝트 생성, 설계 입력, 3D/SVG 편집, Excel/PDF/DXF 출력 또는 Excel Parsing을 수행하는 사용자 흐름을 설명한다.

## 수정된 입력 흐름

```text
공정 설계
  -> 공정별 필요 장비 산출
  -> 기계 카탈로그에서 장비 모델 선택
  -> 카탈로그의 전기 관련 속성 확보
  -> 전기 기기 리스트 자동 생성
  -> 전기 계산 및 성과품 생성
```

전기설계 입력값은 사용자가 전기계산서에 모두 직접 입력하는 구조가 아니라, 공정 설계 결과와 기계 카탈로그 선택 결과에서 자동 생성된다. 사용자는 전기설계 단계에서 MCC 그룹, 예외 기동방식, 수용율 조정, 예비 수량, Tray 경로 같은 설계 판단이 필요한 항목을 검토/수정하는 역할에 가깝다.

## 기능정의서 기준 전기설계 범위

### 계산

기능정의서의 전기 계산 요구사항은 `공종별 계산-전기` 또는 `전기 용량 계산`으로 정의되어 있다.

- 기계 동력 부하에 따른 전기 용량계산서 산출
- 저압반, 고압반 관련 계산 산출
- 부하일람표 산출
- 케이블 스케줄표 산출
- IO 리스트 산출
- 계측기 리스트 산출 및 표출
- 계산 결과를 고정 메뉴에서 조회
- 계산시트를 Excel 또는 PDF로 저장
- 국가별 기준정보와 계산식을 관리자 테이블로 관리

계산서 샘플과 대응되는 산출물:

- `동력설비 부하계산서`
- `부하일람표`
- `동력설비 리스트`
- `계측기리스트`
- `SBR IO MCC`
- `SBR IO 계측제어`
- `IO 집계표`
- `케이블스케쥴(동력)`
- `케이블스케쥴(제어)`
- `케이블스케쥴(계측)`

### 3D 연계

기능정의서에는 전기 계산 결과를 3D 배치와 연결하는 요구사항이 별도로 존재한다.

- 전기실 판넬 배치
- 전기 용량 계산서에서 산출된 판넬 외함 규격을 3D 모델에 적용
- MCC 구성도에서 반영된 외함 규격을 3D 모델에 반영
- 전기 Tray 경로 설정 및 조정
- 전기 용량계산서 기반 Tray 규격 자동 적용
- Tray 경로를 2D 도면으로 산출
- 배관과 Tray 간섭 체크
- Tray의 From-To 정보를 기반으로 동력/제어 케이블 길이 산출

계산서 샘플과 대응되는 산출물:

- `케이블 트레이 계산 산출서(전기실 제외)`
- `케이블 트레이 계산 산출서(전기실)`
- `배전반(전선&차단기) (성과품 X)`

### 2D 도면 생성

전기파트는 기능정의서에서 `SFR-07` 계열로 별도 정의되어 있다.

- 부하일람표 생성
- 수변전설비 단선결선도 생성
- MCC 단선결선도 생성
- MCC 구성도 생성
- 전기 평면도 생성
- 케이블 스케줄표 생성
- PDF/DXF 저장
- 도곽, 심볼, 테이블, 도면 스타일 관리

계산서 샘플은 아직 단선결선도나 MCC 구성도 도면 자체를 포함하지 않고, 그 도면을 생성하기 위한 계산/리스트/케이블/IO 데이터를 제공하는 형태다.

### 물량 및 내역

전기 물량/내역 요구사항은 전기 용량계산과 Tray 데이터를 연동한다.

- 고압반, 저압반, MCC 등 전기 항목 물량 산출
- 수변전설비, MCC 등 항목별 단가 적용
- 전기 Tray 길이 등 물량 산출
- 전기 케이블 길이 산출
- Excel 내역서 양식 반영
- 단가 관리 DB 연동 및 수정

## 튜토리얼 매뉴얼 기준 사용자 흐름

PDF 텍스트 추출 시 일부 한글이 누락되지만, 전기설계와 연결되는 흐름은 다음과 같이 확인된다.

- 프로젝트 생성 후 설계 조건을 입력한다.
- 3D Layout에서 MCC, TrayH, TrayV 같은 전기 관련 3D 요소를 배치한다.
- 전기설계 화면에서 기본 설정값을 입력한다.
  - 예: 여유율 20%
  - 예: MCC 기준 전류 400A
  - 예: 건축부하 비율 또는 MCC 관련 기준값
- Equipment 정보 변경은 3D와 전기설계에 연동된다.
- Excel 다운로드 기능이 제공된다.
- Excel Parsing 기능으로 외부 Excel을 다시 불러오는 흐름이 있다.
- SVG 기반 도면은 PDF/DXF로 출력할 수 있다.
- DXF 출력에는 일반 도면, MCC 관련 도면, 3D 관련 도면 출력 흐름이 포함된다.

## 프로그램 모듈 제안

### 1. Input Registry

입력 원장을 명확히 분리한다. 특히 Equipment는 기계 카탈로그 선택 결과에서 자동 생성되는 중간 데이터로 본다.

- Process design result
- Mechanical catalog selection
- Equipment generated from catalog
- Instrument
- Project electrical settings
- MCC grouping settings
- Tray routing settings
- Output/export settings

### 2. Reference Data

Excel 참조 시트는 DB 또는 설정 파일로 분리한다.

- CV/GV 선정 기준
- 전선 외경
- 차단기 선정
- MC/EOCR/인버터/콘덴서/소프트스타터 선정
- MCC 사이즈
- PLC 전선 및 신호 코드
- 계측기 전선/PLC 전선 기준
- 전선관 선정 기준
- 국가별 기준정보
- 단가/물량 기준정보

### 3. Calculation Engine

Excel 수식을 코드화하되, 수식은 관리자 기준정보와 버전으로 관리한다.

- 부하용량 계산
- 정격전류 및 상용전류 계산
- MCC 그룹별 부하 집계
- 수용율 적용
- 전선/접지선/차단기/제어기기 선정
- IO 산출 및 여유율 적용
- 케이블 스케줄 생성
- 전선관 선정
- Tray 규격 및 물량 산출

### 4. Drawing Data Generator

2D/3D 도면 생성 전에 중간 데이터를 표준화한다.

- Load schedule table
- Single line diagram data
- MCC unit layout data
- Panel placement data
- Tray route data
- Cable schedule table
- IO table

### 5. Export/Import

성과품 입출력을 명확히 분리한다.

- Excel export
- PDF export
- DXF export
- Excel parsing/import
- SVG drawing export
- 출력 대상과 `성과품 X` 내부 검토 자료 구분

## 계산서 샘플과 요구사항 간 Gap

- 계산서 샘플에는 단선결선도, MCC 구성도, 전기 평면도 자체는 없다. 대신 이 도면을 만들기 위한 데이터가 있다.
- 기능정의서에는 국가별 기준정보, 관리자 계산식 관리, 단가 DB가 요구되지만 계산서 샘플에는 Excel 참조 시트 형태로만 들어 있다.
- Tray 경로 설정, 간섭 체크, 케이블 길이 산출은 기능정의서와 매뉴얼에 존재하지만 계산서 샘플은 주로 케이블 외경 합계와 Tray 선정 중심이다.
- Excel Parsing은 매뉴얼에 기능 흐름이 있으나, 어떤 Excel schema를 받아야 하는지는 별도 정의가 필요하다.
- 전기설계 UI 입력값은 매뉴얼에 일부 보이지만, 계산서 샘플에서는 `전기 설계 UIUX(성과품X)` 시트에 초안 수준으로 남아 있다.
