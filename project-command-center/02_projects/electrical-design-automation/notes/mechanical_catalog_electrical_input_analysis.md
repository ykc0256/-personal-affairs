## 핵심 정정

전기설계 자동화의 입력 구조는 사용자가 전기 항목을 직접 처음부터 입력하는 방식이 아니다.

기본 흐름은 다음과 같다.

```text
사람이 공정 설계
  -> 공정에 필요한 기계/설비가 결정됨
  -> 사용자가 기계 카탈로그에서 장비 모델 선택
  -> 카탈로그의 동력/기동방식/전압/효율/역률/수용율이 전기설계 입력값으로 전달
  -> 전기 용량계산서, 부하일람표, 전선/차단기/IO/케이블 스케줄 자동 생성
```

따라서 `초기 데이터`는 사용자가 수기로 만든 전기 입력 원장이 아니라, 공정 설계와 기계 카탈로그 선택 결과를 전기계산서 형태로 정규화한 중간 데이터로 봐야 한다.

## 전기설계에 필요한 공통 컬럼

| 컬럼 | 내부 필드 예시 | 전기설계에서의 역할 |
| --- | --- | --- |
| 동력(kW) | `s_power_kW` | 부하용량, 정격전류, 상용동력 산출 |
| 최소 동력(kW) | `i_min_power_kW`, `s_min_power_kW` | 범위형 장비의 선정 조건 또는 하한값 |
| 최대 동력(kW) | `i_max_power_kW` | 펌프/발전설비 등 범위형 장비의 대표 동력 후보 |
| 전기용량(kW) | `i_power_copacity_kW` | 소독설비 등 전기 소비 용량 |
| 기동방식 | `s_ctrl_method` | 직입/인버터/소프트스타터/MOP에 따른 MCC 구성 |
| 정격전압(V) | `s_rated_volt_V` | 전류 계산과 장비 전압 기준 |
| 효율(%) | `s_efficiency_percent` | kVA 및 정격전류 계산 |
| 역률(%) | `s_pwr_factor_percent` | kVA 및 정격전류 계산 |
| 수용율(%) | `s_demand_factor_percent` | 수용동력/수용부하 계산 |

```text
  - 공정별 필요 장비 산출

기계 카탈로그 선택
  - 장비유형 선택
  - 공급업체/모델 선택

  - 상용/예비 수량 반영
  - MCC 그룹 및 조작반 조건 반영

  - 부하용량, 정격전류, 상용전류 산출
  - 부하일람표 산출
  - 전선, 차단기, 제어기기 선정
  - IO 및 케이블 스케줄 산출

성과품 생성
  - 전기 용량계산서
  - 부하일람표

  - 단선결선도/MCC 구성도/전기 평면도
```

## 설명자료에 반영할 문구

> WAI Design의 전기설계 자동화는 전기 입력 화면에서 모든 값을 수기로 입력하는 구조가 아니다. 사용자가 공정을 설계하고, 공정별 장비를 기계 카탈로그에서 선택하면 카탈로그의 동력, 기동방식, 정격전압, 효율, 역률, 수용율이 전기설계 입력값으로 자동 전달된다. 전기계산서는 이 기계 카탈로그 기반 입력을 KEC/KS/제조사/사내 기준정보와 결합해 용량계산서, 부하일람표, 전선/차단기 선정, IO, 케이블 스케줄, 도면 산출물로 확장하는 단계다.

## 검토 포인트

- 카탈로그의 효율 값이 일부 시트에서 `0.865` 형태와 `86.5` 형태로 혼재할 수 있다. 전기계산 전에 `%` 값의 정규화가 필요하다.
- 펌프처럼 `최소 동력`과 `최대 동력`이 있는 장비는 어떤 값을 전기 부하 대표값으로 사용할지 명확히 해야 한다.
- 소독설비처럼 `전기용량(kW)`과 `동력(kW)`이 함께 있거나 일부만 있는 장비는 부하 계산 기준을 별도로 정해야 한다.
- 카탈로그 코드값 `S_SYS01` 등은 출력용 표시명과 계산용 코드값을 분리 관리하는 것이 좋다.
- 공정 설계 결과에서 상용/예비 수량이 결정되어야 전기 부하계산서의 수량과 상용전류가 확정된다.
