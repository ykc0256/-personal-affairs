```
  └─ INSTRUMENT 조건 조회

```

## 2. 전기 수량산출 및 케이블 관련

### 케이블트레이 수량 산출 이슈 (2026-03-25, 김정선)
- WAI Design(Unity)에서 생성되는 모든 전기 트레이는 **W500mm × H100mm 2단 적재** 형태
- ELBOW, TEE 타입 없이 **직관 형태**만 생성
- 내역서 기재 방식 문제: **길이(m)**로 산출할지 **개수**로 산출할지 미결
- 관련 파일: `전기 수량 산출_r2.xlsx` 중 `2-3. 케이블트레이(개발중)` 시트

> ⚠️ **현재 미결 이슈**: 케이블트레이 수량 산출 단위 (길이 vs 개수) 확인 필요

### 전선관 기준거리 vs 전선관 길이 처리 (2026-03-20, 김성철)
- 동일 기계가 여러 대인 경우, 전선관 길이와 기준거리의 최장거리 장비가 **불일치**하는 상황 발생
- 두 가지 안:
  - **안 1**: 기준거리 기준 최장거리 장비 값 적용
  - **안 2**: 전선관 길이 기준 최장거리 장비 값 적용
- 이전 구두 협의: "가장 긴 거리 사용"으로 방향 설정되었으나 명시적 결정 미확인

### 전선관/면수 산출 API (2026-03-31, 김성철)
- **동력설비 리스트 API** 호출 시 전선관 관련 데이터 반환
  - 전선관 사이즈 (동력/제어 구분)
  - 전선관 길이
  - MCC 면수
- 첨부: `동력설비 리스트-api.json`

### 전기 수량산출 파일 버전 이력
| 파일명 | 시점 | 비고 |
|--------|------|------|
| `전기 수량 산출_r2.xlsx` | 2026-03 | 케이블트레이 시트 개발중 포함 |
| `260401_전기 수량 산출_r3.xlsx` | 2026-04 | 최종 분석 기준본 |
