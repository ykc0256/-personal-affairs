## 전체 탭 구조 및 작업 순서

WAI Design은 상단 탭 순서대로 작업을 진행한다.

```
[프로젝트] → [공정 계획] → [3D 레이아웃] → [산출물]
```

| 순서 | 탭 | 주요 작업 |
|------|-----|----------|
| 1 | 프로젝트 | 신규 프로젝트 생성, 단위체계 선택(METRIC/USCS), 부지도면 SVG 업로드 |
| 2 | 공정 계획 | 공정 블록 구성, 전기 조건 설정, 전기 도면 생성 |
| 3 | 3D 레이아웃 | 구조물·기계 배치, 배관 라우팅, 전기 트레이 배치, 도면 생성 |
| 4 | 산출물 | 계산서·내역서·도면 다운로드 |

## 공정 계획 탭 — 전기 관련 흐름 (5.7, 5.8)

### 5.1 ~ 5.6: 전기 계산의 입력 준비

```
5.1 프로세스       공정 블록 추가·연결
5.2 공정 시나리오  시나리오 설정
5.3 검토          결과 검토
5.4 용량 계산서    공정·기계 용량 계산 결과
5.5 기계 리스트    장비 목록 확정 → 전기 입력값 원천
5.6 P&ID          P&ID 생성
```

### 5.7 전기 설계 ← 전기 계산의 핵심 단계

전기 설계 모드를 클릭하면 설정 팝업이 열린다.

**수변전 설비 유무 선택**

| 선택 | 동작 |
|------|------|
| 있음 | 저압반 + 고압반 전기설계 진행, 내역서에 수변전설비 시트(8-1, 8-2) 포함 |
| 없음 | 저압반 전기설계만 진행, 수변전설비 시트 제외 |

**전기 조건 설정**

| 항목 | 기본값 | 유효 범위 | 설명 |
|------|--------|----------|------|
| 건축 부하 | 20% | 0 초과 ~ 100 이하 | 동력설비 부하 대비 건축 부하 적용율 |
| MCC별 한계 상용 전류 | 400A | 0 초과 | MCC 그룹의 상한 전류값 |
| 건축 설비 분전반 수용율 | 100% | 0 초과 ~ 100 이하 | 건축 부하의 수용율 |

**저장 후 자동 생성되는 산출물**

```
저장
 ├─ 동력설비 부하계산서  (동력설비 탭에서 확인)
 └─ 부하일람표          (부하일람표 탭에서 확인)
```

> Note: 전기 조건 설정 이후 Equipment(기계)를 변경하면 동력값이 달라지므로, 반드시 전기 조건 설정값을 다시 저장한 뒤 3D 레이아웃으로 이동해야 한다.

### 5.8 전기 도면 ← 전기 조건 설정 완료 후 활성화

```
전기 도면 모드 진입
 → 도면 항목별 [SVG 생성] 버튼 클릭
 → SVG 편집기에서 도곽(Border) 설정
 → [Export as PDF & DXF] 내보내기
```

**도면 내보내기 방식**

| 탭 | 도면 종류 | 내보내기 방식 |
|----|----------|--------------|
| 공정 계획 탭 (일반) | 부하일람표, 수변전 단선결선도 등 | 단일 SVG → 단일 PDF/DXF |
| 공정 계획 탭 (MCC 구성도) | MCC 구성도 | 복수 SVG → 단일 DXF |
| 3D 레이아웃 탭 | 배치 도면, 단면도, 평면도 등 | SVG 파일 수만큼 DXF |

> Note: SVG 편집기에서 도면 수정 후 반드시 PDF 또는 DXF로 내보내기해야 산출물 탭에서 확인 가능하다.

## 3D 레이아웃 탭 — 전기 트레이 배치

3D 레이아웃 탭은 공정 계획 결과를 바탕으로 DB에서 3D 모델을 로딩한다.

### 모드 순서 (위→아래 순으로 작업)

```
구조물/기계 편집   구조물 편집, 기계 배치
적층/오픈홀        기초 콘크리트 생성, 토목 수량 산출
배관 라우팅        배관 생성, 프리셋 불러오기
전기 트레이  ←    전기 트레이 배치 및 전선 연결
수리계통도 생성    배치 기반 수리계통도 생성
도면 생성          배치 기반 도면 생성
```

### 전기 트레이 배치 세부 흐름

```
[전기 트레이] 모드 클릭
 ├─ MCC 배치
 │    [MCC] 아이콘 클릭 → MCC 생성
 ├─ Tray 배치
 │    [TrayH] → 수평 트레이 생성
 │    [TrayV] → 수직 트레이 생성
 │    객체 편집 메뉴로 회전·길이 설정
 └─ 전선 연결
      [커넥터] 아이콘 → 트레이 위 위치 선택 → C키 → 커넥터 생성
      시작·끝 커넥터 선택 → [전선 생성] 클릭 → 전선 연결
```

## 산출물 탭

공정 계획·3D 레이아웃에서 생성·Export한 파일을 다운로드하는 탭.

```
계산서   동력설비 부하계산서, 부하일람표 등
내역서   전기 기자재내역서, 설치비내역서 등
도면     Export된 PDF·DXF 목록
```

---

## 관리자 워크플로

사용자가 완료 → 승인 요청 후, 관리자는 다음 순서로 검토한다.

```
1. 로그인 (관리자 계정)
2. 프로젝트 열기
3. 산출물 탭 → 계산서·내역서·도면 다운로드 검토
4. 3D 레이아웃 탭 → 배치 검토
   (주의: 공정 계획으로 이동하면 3D 레이아웃이 초기화됨)
5. 공정 계획 탭 → 계획 검토
6. 승인 또는 반려
```

## 전기설계 흐름 전체 요약

```
[공정 계획 탭]
  기계 리스트 확정 (장비 동력·기동방식 원천)
       ↓
  5.7 전기 설계
    수변전 설비 유무 선택
    전기 조건 설정 (건축부하, MCC 한계전류, 수용율)
    저장
       ↓
  자동 생성
    동력설비 부하계산서
    부하일람표
       ↓
  5.8 전기 도면
    SVG 생성 → 도곽 설정 → PDF/DXF 내보내기
    (부하일람표, 수변전 단선결선도, MCC 단선결선도, MCC 구성도 등)

[3D 레이아웃 탭]
  전기 트레이 모드
    MCC 배치
    TrayH / TrayV 배치
    전선 커넥터 생성 → 전선 연결
       ↓
  도면 생성 → SVG → PDF/DXF

[산출물 탭]
  계산서 다운로드
  내역서 다운로드
  도면 다운로드 (PDF/DXF)
```
