# Deliverable Checklist

## Common

- [ ] 프로젝트명 또는 공사명이 식별된다.
- [ ] 도면번호, 문서번호, Revision이 확인된다.
- [ ] 작성일 또는 최종 수정일이 확인된다.
- [ ] 파일명이 정리 기준에 맞는다.
- [ ] 중복 파일 여부를 확인했다.

## Electrical Design

- [ ] 전기계산서
- [ ] 도면 목록
- [ ] 단선결선도
- [ ] 배치도
- [ ] 배관/배선도
- [ ] 부하 계산 관련 자료
- [ ] 수량산출서 또는 내역 관련 자료
- [ ] 장비 사양서 또는 카탈로그
- [ ] 준공도서
