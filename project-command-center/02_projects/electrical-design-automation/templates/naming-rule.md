# Naming Rule

초기 파일명 규칙 초안입니다. 실제 성과품을 검토하면서 조정합니다.

```text
YYYYMMDD_Project_Discipline_DocumentType_DocumentNo_Rev.ext
```

## Examples

```text
20260512_ProjectA_ELEC_Calculation_Load_R00.xlsx
20260512_ProjectA_ELEC_Drawing_SLD_R01.dwg
20260512_ProjectA_ELEC_Closeout_Index_R00.xlsx
```

## Document Type Candidates

- `Calculation`
- `Drawing`
- `Spec`
- `Catalog`
- `Quantity`
- `Closeout`
- `Reference`
