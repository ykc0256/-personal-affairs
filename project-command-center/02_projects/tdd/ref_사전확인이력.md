## 3. 내역서(BOQ) — 수량 산출 관련

출처: `TDD_내역서_미확정항목_확인용.md` (구 파일명)

### 방향 결정 항목

**A-1. 건축 BOQ 담당 체계**

→ 건축물의 경우 3D 환경에서 건축물을 배치할 경우 해당 건축물의 면적이 전달되는 구조. TDD 범위에 포함. 내역서(양식)에 있는 항목들을 전부 산출해야 함

**A-2. 배관 내역서 — 3D 라우팅 없는 프로젝트 처리**

→ 헤더만 있는 빈 시트 생성

**A-3. 케이블트레이 커버 설치 여부**

→ 커버 설치는 없음

**A-4. 전기 케이블 길이 산출 우선순위**

→ 실제 경로 기반이 기본임. 실제 경로의 값이 전기계산서에 반영되는 구조

**A-5. 파생 품목 산식 계수 — 고정값 여부**

→ 모든 프로젝트에 동일. Name Plate의 경우 MCC용과 현장조작반용 구분 필요. ×3, ×1 둘 다 있음

**A-6. 토목 내역서 수량 산출 공식**

→ 3D 환경에서 토목 수량 산출 관련 내용 지정 시 나오는 데이터. 메뉴얼 확인 바람

**A-7. 설치비 내역서 단가 출처 (기계 파트)**

→ 장비에 호표 번호가 작성되어 있음. `s_primary_item_no_NONE` 형태. 없으면 설치비 없음
