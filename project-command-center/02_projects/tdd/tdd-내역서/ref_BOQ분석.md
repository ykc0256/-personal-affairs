## Part 1. 배관 내역서 산출과정 자료 분석

목적: 내역서 TDD 작성 시 배관 및 전기 연계 내역 산출 규칙 참고용.

### 1-1. 자료 성격

파일명은 배관 산출과정 정리이지만 실제로는 배관+전기 범위를 함께 포함한다.

| 구분 | 포함 여부 | 설명 |
|---|---:|---|
| 배관 산출 | 포함 | 배관 카탈로그, 3D 산출 결과, 설치비 호표 연결 구조 |
| 전기 배관/배선 | 포함 | 케이블, 접지선, 전선관, 가요전선관, 단자류 |
| 케이블트레이 | 포함 | 트레이 본체, 커버, 엘보, 티, 행거/부속품 |
| 전동기제어반 | 포함 | MCC, LOP, 차단기, SPD, CT, MC 등 |
| 현장조작반 | 포함 | LOP 수량 기반 부속품 산출 |
| 최종 내역서 양식 | 포함 | `5-1. 배관공사비(탬플릿)` 시트에 금액 계산 구조 존재 |

### 1-2. 시트 구조

| 시트 | 역할 |
|---|---|
| `배관산출과정 정리` | 카탈로그, 일위대가목록, 3D 산출 결과 매핑 |
| `5-1. 배관공사비(탬플릿)` | 내역서 출력 템플릿 (금액 = 수량 × 단가) |
| `나올수있는항목(전기)` | 전기 내역 후보 전체 목록 |
| `단자 산출` | 케이블 규격별 단자 수량 산출 |
| `전선관 산출` | 전선관·가요전선관·박스커넥터·노말밴드 산출 |
| `케이블트레이 산출` | 트레이 본체/부속품 산출 |
