### Q-1. F-CV 2.5SQ 3C 설치비 0원 — 의도적 규칙인가?

**확인된 사실:**  
GWD BOQ `6-2. 설치비내역서(전기)` 에서 F-CV 2.5SQ 3C 행:
```
수량=150M, 합계=0원, 노무비=0원, 경비=0원
```
다른 케이블(F-CV 4SQ 3C 등)은 모두 노무비/경비가 존재함.

**질문:**  
소형 케이블(2.5SQ 이하) 또는 특정 용도 케이블은 설치비를 별도 산정하지 않는 규칙이 있나요?  
아니면 GWD 프로젝트에서만 예외적으로 0원인 건가요?

→ 답변: DB 안에 호표에 대한 내용이 없는 제품들의 경우, 설치비가 확정되지 않은 내용임. 즉, 호표를 기준으로 산출되는 방식을 그대로 반영하면 됨. 해당 내용 추가는 관리자의 몫이기 때문에 이점 참고

> 다음 액션: 규칙이 명확하면 TDD 내역서 3-6에 주석 추가. 불명확하면 개발팀 소스코드 확인 요청(BOQ-2 연계).

+ M-202(1.3kW, 수용60%) + M-203(0.4kW×2, 수용80%) + M-204(1.5kW, 수용60%)

32.62kW와 30.25kW는 어떻게 다른 값이 나오는 건가요?  
계약전력과 부하일람표 수용동력은 각각 어떤 의미로 다르게 사용되나요?  
(예: 계약전력 = 건축설비 분전반 포함? 또는 수용율 적용 방식 차이?)

→ 답변:계약전력의 경우, 건축부하 % 설정값에 따라 건축부하에 대한 값도 포함된 상황.
