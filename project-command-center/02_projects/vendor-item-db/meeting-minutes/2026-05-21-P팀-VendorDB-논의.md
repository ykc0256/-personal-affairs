## 논의 내용

### P팀 현황

- 현재 MS365를 DB 대체 수단으로 사용 중이나 데이터 정리가 어려운 상황
- MS365로는 협업이 어려워 DB 전환 필요성을 인식하고 있음
- 현재 민수/TK에 대한 SPEC 정리 중

### GWD 연계 어려운 이유

- **사용 목적 차이**: TSK의 사용 목적은 실행검토, 기자재 spec별 datasheet 정리, 프로젝트별 기자재 DATA 정리
- **업무 구조 문제**: PS에서 설계 → TSK로 이전하는 구조. GWD가 PS 단계에서 사용되어야 DB 공유 접점 발생
- **데이터 불안정성**: EPC팀에서 정리 중인 내용을 WAI팀이 파악하기 어렵고, EPC팀 자체도 처음 정리하는 중이라 컬럼이 계속 변경됨
