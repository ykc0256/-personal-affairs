## 반영한 수정사항

| 구분 | 수정 내용 |
|---|---|
| 로그인 | `로그인 아이디 저장` 체크박스 추가. 선택 시 브라우저 localStorage에 ID 저장 |
| GNB/문구 | 주요 메뉴와 로그인/대시보드/관리 화면의 깨진 한글 문구 일부 복구 |
| 기자재 목록 | 좌측 `equipment_categories` 기반 기계 분류 필터 추가 |
| 기자재 목록 | 분류, 기자재, 단위, 취급업체 수, 설계가, 실행가, 평가 기준 오름차순/내림차순 정렬 추가 |
| 기자재 목록 | admin/procurement는 설계가+실행가 표시, 그 외 권한은 설계가만 표시 |
| 기자재 상세 | `/equipments/[id]` 추가. 기본정보, 취급 업체, 평가/가격 우선순위 표시 |
| 업체 목록 | 좌측 기계 분류 필터 추가. 선택한 분류의 기자재를 취급하는 업체만 표시 |
| 업체 목록 | 업체, 유형, 국가, 재무등급, 최근 평가, 취급품목 수, 설계가, 실행가 기준 정렬 추가 |
| 업체 상세 | `/vendors/[id]` 추가. 기본정보, 최근 평가, 취급 기자재와 가격 표시 |
| 단가 비교 | 좌측 기계 분류 필터 추가 |
| 단가 비교 | admin/procurement는 설계가+실행가 표시, 그 외 권한은 설계가만 표시 |
