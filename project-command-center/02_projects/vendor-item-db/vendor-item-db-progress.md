## 현재 구조 판단

| 항목 | 판단 |
| --- | --- |
| 분류 구조 | 기자재와 업체가 같은 `기계 / 배관 / 전기` 분류 트리를 공유하는 구조는 맞다. |
| 분류 UX | 분류명 클릭으로 선택하고 색상으로 표시하는 방식이 맞다. 별도 `선택` 버튼은 제거하는 편이 낫다. |
| 화면 이동 | 기자재와 업체 화면 사이에서 같은 분류 선택 상태를 유지하는 구조가 맞다. |
| 기자재 상세 | 기자재명 클릭 시 단가 변경 이력만 보여주는 인라인 확장 구조가 맞다. |
| 단가 관리 | 단가 비교 페이지로 이동시키는 것은 맞지 않다. 기자재 확장 패널 내부에서 입력/정정하는 구조가 맞다. |
| 단가 이력 | `기준일 + 업체` 기준으로 설계가/실행가를 한 행에 묶는 구조가 맞다. |
| 업체 상세 | 업체명 클릭 시 평가 이력과 보유 기자재를 같은 방식으로 아래 확장 패널에 두는 구조가 맞다. |
