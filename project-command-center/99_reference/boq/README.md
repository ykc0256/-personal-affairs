# BOQ Reference Materials

내역서(BOQ) 산출 과정, 원본 양식, 수량 산출 예시를 보관하는 참조자료 폴더입니다.

## Originals

| 파일 | 용도 |
|---|---|
| `originals/251217_내역서 산출과정 정리(배관)_r1.xlsx` | 배관 내역서 산출 과정 정리. 배관뿐 아니라 전기 내역 항목, 전선관, 케이블트레이, 전동기제어반, 현장조작반 산출 예시를 포함한다. |

## Usage

- TDD 작성용 원천 자료로만 사용한다.
- 구현 문서에는 원본 파일의 시트 구조, 산출식, DB/API 연결 필요 항목만 요약해 반영한다.
- 원본 Excel의 수식은 업무 규칙 후보로 보고, 최종 규칙은 소스코드/API/DB와 대조 후 확정한다.
