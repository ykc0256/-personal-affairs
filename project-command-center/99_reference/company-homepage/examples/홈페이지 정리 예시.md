# 1. 홈페이지 구성 정리

## 1.1 기술소개 항목

**: 기술소개는 간단하게 PROTEUS에 대한 기술 개요 및 이미지 반영**

**: 기술소개 후 사업분야 구분 및 링크 반영**  
→ 대분류는 "수처리"와 "재이용" or 기존 사업분야 항목 열거

ㄴ 수처리 → 일차처리, 주처리, 간이공공  
ㄴ 재이용 → 하수재이용, RO농축수처리

※ PROTEUS의 대표 특장(여재 반영된 내용이면 GOOD) 내용 반영 필요

## 1.2 사업분야 항목

- 기존 항목 및 내용 유지 → 하수 재이용, 일차처리(패키지 추가), 간이공공(패키지 추가), RO농축수

**: 기존 항목 추가사항 → 주처리(Secondary Treatment) 추가 필요(?)**

- 각 분야별 적용 기술소개자료 링크

ㄴ 하수 재이용 → Tertiary Treatment & Water Reuse  
ㄴ 일차처리 → Primary Treatment + (패키지 추가)  
ㄴ 간이공공하수처리 → Wet Weather Flow Treatment + (패키지 추가)  
ㄴ RO농축수처리 → RO Concentrate Treatment  
ㄴ 주처리 → Secondary Treatment ▶ 추가시 반영

※ 기존 통합된 기술소개자료를 각 사업분야 별로 파일 분류 필요
