# 국내 전기설계 기준 Reference Map

목적: `260106_전기계산서(최종샘플).xlsx`의 `(참조)` 시트들이 어떤 국내 기준, KS/IEC 표준, 제조사 카탈로그, 사내 설계 기준에 근거해야 하는지 추적하기 위한 기준 맵입니다.

이 문서는 상세 설명자료와 프로그램 기준정보 설계에 사용할 수 있도록 작성했습니다. 실제 허용전류표, 차단기 정격표, 제조사 모델표 등은 저작권 및 개정 이슈가 있으므로 원문 표를 그대로 복제하지 않고, 출처와 검증 포인트 중심으로 관리합니다.

## 기준 체계

국내 전기설계 기준은 다음 우선순위로 보는 것이 적절합니다.

1. 법령/고시
   - 전기사업법 제67조 및 시행령 제43조에 근거한 전기설비기술기준
   - 한국전기설비규정(KEC)

2. KEC 및 KEC 핸드북
   - 저압 전기설비
   - 접지시스템
   - 배선설비
   - 과전류 보호
   - 전선 및 보호도체 선정

3. KS/IEC 표준
   - KS C IEC 60364 계열
   - KS C IEC 60364-5-52: 배선설비
   - KS C IEC 60364-5-54: 접지설비 및 보호도체
   - KS C IEC 61537-A: 케이블 트레이 및 케이블 래더 시스템

4. 제조사 카탈로그
   - 차단기, 전자접촉기, EOCR, 인버터, 소프트스타터, 콘덴서 등 모델 선정
   - 제조사별 모델명, 정격, 적용 전동기 용량, 단락용량, 프레임, 외형 치수

5. 프로젝트/발주처/사내 기준
   - 수용율
   - MCC 그룹 한계 전류
   - Spare/여유율
   - 건축부하 반영 비율
   - Tray 여유율
   - 표준 전선/전선관/케이블 Tray 선호 규격

## 공식 출처

### 전기설비기술기준

- 출처: 대한전기산업연합회 전기설비기술기준 플랫폼
- URL: https://kec.kea.kr/sub_about/overview.php
- 확인 내용:
  - 전기설비기술기준은 전기사업법 제67조 및 시행령 제43조에 근거한다.
  - 전기설비의 설계, 시공, 인허가, 사용전 검사에 필요한 최소 안전기준으로 설명된다.

### 한국전기설비규정(KEC)

- 출처: 대한전기산업연합회 전기설비기술기준 플랫폼
- URL: https://kec.kea.kr/sub_about/regulation.php
- 확인 내용:
  - KEC는 국제표준(IEC)을 기초로 국내 환경에 맞게 정리된 전기규정이다.
  - 2021년 1월 1일부터 기존 `전기설비기술기준의 판단기준`을 대체했다.
  - 과전류보호 등 상세사항을 객관 지표 기반으로 부합화하는 목적을 가진다.

- 국가법령정보센터 KEC 검색: https://www.law.go.kr/admRulSc.do?menuId=5&query=%ED%95%9C%EA%B5%AD%EC%A0%84%EA%B8%B0%EC%84%A4%EB%B9%84%EA%B7%9C%EC%A0%95

- 출처: e나라 표준인증

- 확인 내용:
  - 케이블과 전기설비를 지지하고 수용하는 케이블 트레이/래더 시스템의 요구사항과 시험 기준이다.
  - Tray 자체 제품 기준 또는 성능 기준 검토에 해당한다.

## 참조표별 보강 방향

### 1. CV/GV 선정

현재 계산서는 상용전류를 기준으로 CV 전선과 GV 접지선을 단순 매칭한다.

보강해야 할 조건:

- 전압: 저압 3상 380 V 기준인지 명확화
- 도체 재질: 동도체 기준인지 명확화
- 절연체 종류: CV, F-CV 등 구분
- 코어 수: 3C, 4C 등 구분
- 포설 방식: 전선관, 트레이, 공기 중, 지중 등
- 주위온도 보정
- 다회로/집합 포설 보정
- 전압강하 검토
- 단락 시 열적 내량 검토
- 보호도체 단면적 선정 기준

설명자료 문구:

> 전선 선정은 단순히 부하전류보다 큰 허용전류를 갖는 규격을 고르는 것이 아니라, 포설조건, 주위온도, 회로 집합, 전압강하, 단락내량, 보호도체 기준을 함께 검토해야 한다. 계산서의 `(참조) CV 및 GV`는 자동 산출을 위한 1차 매칭표이며, 최종 설계기준은 KEC 및 KS C IEC 60364-5-52/5-54에 따라 검증한다.

### 2. 전선외경

현재 계산서는 케이블 스케줄과 전선관/Tray 계산을 위해 전선 외경을 참조한다.

보강해야 할 조건:

- 제조사별 외경 차이
- CV/F-CV/CVV/CVV-SB 등 케이블 타입 구분
- 코어 수와 단면적별 외경
- 동일 규격이라도 제조사/차폐/난연/저독성 여부에 따른 외경 차이

설명자료 문구:

> 전선 외경은 KEC의 계산식만으로 고정되는 값이 아니라 제조사 카탈로그 값에 의존한다. 따라서 프로그램 기준정보에는 제조사, 제품군, 공칭단면적, 코어 수, 차폐 여부, 외경, 적용일자를 함께 저장해야 한다.

### 3. 차단기 선정

현재 계산서는 상용전류 구간에 따라 차단기 모델을 매칭한다.

보강해야 할 조건:

- 설계전류와 차단기 정격전류 관계
- 전선 허용전류와 보호장치 정격의 관계
- 차단용량(kA)
- 극수
- 전압
- MCCB/ELCB/MCB/ACB 구분
- 단락전류 계산
- 제조사 모델별 프레임과 정격

설명자료 문구:

> 차단기 선정표는 부하전류 기준의 모델 매칭표로 출발할 수 있지만, 최종 선정에는 전선 보호 조건과 차단용량 검토가 포함되어야 한다. 자동화 프로그램에서는 `정격전류`, `프레임`, `극수`, `차단용량`, `적용전압`, `제조사 모델`을 분리 저장해야 한다.

### 4. MC/EOCR/인버터/소프트스타터

현재 계산서는 기동방식과 전동기 용량을 기준으로 제어기기를 선정한다.

보강해야 할 조건:

- 기동방식: 직입, 인버터, 소프트스타터, MOP 등
- 전동기 용량(kW)
- 정격전류(A)
- 사용전압
- 부하 종류
- 운전 빈도
- 제조사 모델별 적용 범위
- 제어전원 전압
- 보조접점/통신 옵션

설명자료 문구:

> 제어기기 선정은 법규 표준보다는 제조사 카탈로그와 프로젝트 표준의 영향이 크다. 따라서 자동 산출표에는 적용 기준 버전, 제조사, 시리즈명, 모델명, 적용 용량, 적용 전류 범위, 옵션 조건을 포함해야 한다.

### 5. PLC 전선 및 계측 전선

현재 계산서는 기기/계측기 코드별로 DI, DO, AI, AO, RS-485, Ethernet 수량과 케이블 규격을 매칭한다.

보강해야 할 조건:

- 기기 타입별 표준 제어 신호
- 계측기 타입별 전원/신호 방식
- 2-wire/4-wire 구분
- AI 4-20mA, DI 적산, 통신 방식
- 차폐 케이블 필요 여부
- PLC I/O 카드 점수 산출 기준
- 20% spare 적용 기준

설명자료 문구:

> PLC 및 계측 케이블 표는 KEC보다는 제어 표준과 계측기 사양에 가까운 기준정보다. 단, 케이블 포설 및 보호는 KEC/KS 기준으로 검토하고, 신호 수량과 케이블 타입은 사내 표준/계측기 사양으로 관리한다.

### 6. 전선관 선정

현재 계산서는 케이블 외경으로 단면적을 계산하고 전선관 규격을 선택한다.

보강해야 할 조건:

- 전선관 종류: 후강전선관, 박강전선관, 합성수지관 등
- 케이블 수량
- 케이블 외경
- 점유율/충전율 기준
- 향후 증설 여유
- 굴곡/인입 용이성
- 발주처 표준 규격

설명자료 문구:

> 전선관 선정은 케이블 단면적 합계와 점유율 기준으로 결정된다. 계산서의 `(참조) 전선관`은 단면적 기준 매칭표로 볼 수 있으며, 실제 설계에서는 전선관 종류와 포설 조건, 케이블 인입성, 증설 여유를 함께 검토한다.

### 7. Tray 선정

현재 계산서는 전기실/전기실 제외 Tray 산출서에서 케이블 외경 합계를 기준으로 Tray 규격을 산정한다.

보강해야 할 조건:

- 동력/제어/계측 케이블 분리 기준
- Tray 폭/높이/점유율
- 케이블 단층 포설 여부
- 허용하중
- 지지간격
- 부식환경
- 접지 및 본딩
- 방화구획 관통부 처리
- KS C IEC 61537-A 제품 기준

설명자료 문구:

> Tray 계산은 케이블 외경 합계만으로 완료되지 않는다. 케이블 분리, 허용하중, 지지간격, 부식환경, 접지/본딩, 유지보수 여유가 함께 고려되어야 하며, Tray 제품 자체는 KS C IEC 61537-A 또는 제조사 시험자료로 검증한다.

## 기준정보 DB 설계 권장 필드

모든 참조표에는 다음 필드를 공통으로 두는 것이 좋다.

| 필드 | 설명 |
| --- | --- |
| `rule_id` | 기준정보 고유 ID |
| `rule_group` | 전선, 차단기, 전선관, PLC 전선 등 그룹 |
| `standard_source` | KEC, KS, 제조사 카탈로그, 사내 기준 등 |
| `source_title` | 기준 문서명 |
| `source_version` | 기준 버전 또는 발행일 |
| `source_url` | 출처 URL |
| `effective_from` | 적용 시작일 |
| `effective_to` | 적용 종료일 |
| `input_condition` | 적용 조건 |
| `output_value` | 선정 결과 |
| `priority` | 복수 기준 충돌 시 우선순위 |
| `review_status` | 초안, 검토중, 승인, 폐기 |
| `reviewed_by` | 검토자 |
| `reviewed_at` | 검토일 |
| `remarks` | 비고 |
