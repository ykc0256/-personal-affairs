# 251203 Procurement팀 정리

원본: `99_reference/vendor-item-db/originals/251203_Procurement팀.txt`

## 문서 성격

Procurement팀 관점의 Vendor 관리, 구매 실행 DB, GWD 포함 희망 기능을 정리한 메모입니다. 이 프로젝트에서는 특히 3번 항목을 GWD 담당 요구사항으로 봅니다.

## 1. Vendor 관리 프로세스

| 항목 | 내용 |
| --- | --- |
| 업체 등록 | Vendor 등록 체계 필요 |
| 정기 평가 | 반기 2회 평가 |
| 상반기 평가 | 재무평가 |
| 하반기 평가 | 사용자 평가 |
| 등록 구분 | 공사업체, 물품업체, 용역업체 |

## 2. 구매 / 실행 DB

| 항목 | 내용 |
| --- | --- |
| 실행 DB | 별도 구성 필요 |
| 담당 예정 | 엔지니어링팀에서 BOM, 단가, 시장가 정리 예정 |
| 실행가 데이터 | 과거 데이터와 현재 진행 데이터를 모두 보관해야 함 |

## 3. GWD 포함 희망 기능

이 항목을 GWD 담당 범위의 핵심 요구사항으로 반영합니다.

| 기능 | 요구 내용 | 프로젝트 반영 |
| --- | --- | --- |
| 조회 | 업체명, 아이템, 대리점 기준 조회 | 검색 조건 및 인덱스 설계 필요 |
| 다운로드 | 조회 결과 다운로드 | CSV/XLSX 내보내기 요구로 관리 |
| 파일 저장 | Vendor 관련 파일 저장 | 첨부 파일 테이블 또는 파일 경로 관리 필요 |
| 평가 | 과거 데이터 저장 필요 | 평가 이력 테이블 필요 |
| 아이템 단가 | Item별 단가 관리 | Item DB와 가격 이력 구조 필요 |
| 업체 아이디 | WAI팀 진행 | GWD와 WAI 기준 ID 매핑 필요 |
| 컬럼 추가 | 실적, 재무건전성, 사용자 평가, 매출액, 자본금 규모 | Vendor 주요 컬럼 후보 |
| Dashboard | 물품, 공사, 용역, 기타 별 업체 비교 검토 테이블 | 업체 유형별 비교 지표 설계 필요 |

## 컬럼 후보

- 업체명
- 업체 ID
- 업체 구분: 물품, 공사, 용역, 기타
- 아이템
- 대리점
- 실적
- 재무건전성: pass / non pass
- 사용자 평가
- 매출액
- 자본금 규모
- 파일 저장 정보
- 평가 이력
- 단가 정보

## 확인 필요

- 업체 ID의 기준 시스템이 WAI인지 GWD인지 확정 필요
- 다운로드 형식: Excel, CSV, PDF 중 우선순위 확인 필요
- 파일 저장 대상: 사업자등록증, 견적서, 평가서, 계약서 등 범위 확인 필요
- Dashboard 비교 기준: 가격, 납기, 평가, 실적, 재무건전성 중 우선 지표 확정 필요
