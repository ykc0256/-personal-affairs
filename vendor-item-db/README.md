# 업체·기자재 단가 DB

업체 정보, 기자재 분류 체계, 견적·실행 단가를 통합 관리하는 사내 웹 애플리케이션입니다.

---

## 접속 방법

| 항목 | 내용 |
|------|------|
| 주소 | `http://192.168.0.142:3000` (사내망 전용) |
| 기본 계정 | ID: `admin` / PW: `admin` |

> 사내 Wi-Fi 또는 유선 네트워크에 연결된 상태에서만 접속 가능합니다.

---

## 화면 구성

```
대시보드 | 기자재 | 업체·평가 | 관리
```

### 대시보드
전체 현황을 한눈에 확인합니다.
- 전체 기자재 수 / 분류 미지정 항목
- 전체 업체 수 / 평가 미등록 업체
- 가격 미등록 기자재 수
- 최근 30일 가격 변경 건수
- 검토 필요 항목 목록 (분류 미지정, 가격 없음, 비활성 등)

### 기자재
기자재 목록 조회 및 등록·수정.
- 분류·코드·모델명·규격 관리
- 업체별 단가 이력 (견적가 / 실행가) 등록 및 조회
- 가격 확정 / 제외 처리

### 업체·평가
업체 등록 및 평가 관리.
- 업체 기본 정보 (코드, 국가, 사업자번호, 재무 등급 등)
- 업체별 납품 기자재 목록
- 업체 평가 점수 및 등급 이력

### 관리 *(admin / procurement 전용)*
- **기자재 분류 관리** — 분류 코드 계층 구조 CRUD, 엑셀 일괄 등록/삭제
- 사용자 관리 *(준비 중)*
- Excel 업로드 *(준비 중)*

## DB 구조

PostgreSQL 스키마: `bkt_vender_db`

```
equipment_categories     기자재 분류 체계 (최대 4단계 계층)
│  category_code         고유 코드 (영대문자·숫자·_ 조합, 예: ELEC_PANEL_LV)
│  parent_category_id    상위 분류 참조 (null = 최상위)
│  depth                 계층 깊이 (1~4)
│
└── equipments           기자재 마스터
    │  equipment_code    고유 코드
    │  equipment_name / model_name / specification / unit
    │
    └── vendor_items     업체-기자재 연결 (M:N)
        │  vendor_id  ──→  vendors
        │
        ├── design_prices      견적 단가 이력 (날짜별)
        └── execution_prices   실행 단가 이력 (날짜별)

vendors                  업체 마스터
│  vendor_code           고유 코드
│  vendor_type / country / financial_grade / business_no
│
└── vendor_evaluations   업체 평가 이력
    └── evaluation_scores  항목별 점수
```

**단가 이력 구조**  
같은 업체·기자재 조합에 날짜별로 복수의 단가를 기록합니다.  
`is_voided = true`인 레코드는 취소 처리된 이력이며, 유효한 최신 단가는 `is_voided = false` 중 가장 최근 `price_date` 기준으로 산출합니다.

## 기자재 분류 코드 규칙

- 영대문자, 숫자, 언더스코어(`_`)만 사용 가능
- 계층 구조 반영 권장: `ELEC` → `ELEC_PANEL` → `ELEC_PANEL_LV`
- 엑셀 양식으로 일괄 등록 가능 (`관리 > 기자재 분류 > 엑셀로 등록`)

엑셀 열 구성:

| 열 | 항목 | 필수 여부 |
|----|------|-----------|
| A | 분류코드 | 필수 |
| B | 분류명 | 필수 |
| C | 상위분류코드 | 선택 (없으면 최상위로 등록) |
| D | 정렬순서 | 선택 (없으면 자동 배정) |
